### 🗄️ 초기 DB 테이블 명세 (v1.0)

#### 1. Users (사용자 자산)
- 캐릭터의 레벨, 경험치, 골드를 서버에서 단독으로 관리합니다.

#### 2. Item_Master (아이템 정보)
- 기획 데이터입니다. (예: 101-단검, 102-철갑옷 등)

#### 3. User_Inventory (인벤토리)
- `instance_id`를 통해 동일한 아이템이라도 강화도가 다른 각각의 객체를 구분합니다.
- 리니지 스타일의 '각인/비각인' 시스템을 위해 `is_tradeable` 필드를 포함합니다.

#### 4. Gold_Logs (재화 로그)
- 모든 골드 변동을 기록하여 비정상적인 수치 조작을 감시합니다.
